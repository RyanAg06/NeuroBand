## Variables del Sistema
- frecuencia_c: Obtendra los niveles de frecuencia carficada mediante el sensor de pulso
- x_giro: Valor X del giroscopio
- y_giro: Valor Y del giroscopio
- z_giro: Valor Z del giroscopio
- bandera_frecuencia: Se activara si los niveles de frecuencia del usuario superan los limites
- bandera_giroscopio: Se activara si se detecta un moviemiento brusco, extra?o
- tiempo_confirmacion: Tomara el tiempo que llevan activadas ambas banderas
- Bandera_alarma: Si el tiempo de confirmacion supera los 3 segundos se activara la alarma
